# Alessandro Petruzzelli - Personal Website

Welcome to the repository for my personal website!

This site serves as a portfolio of my work, research, and passions. It includes:

*   **Academic Profile:** My research interests, publications, and teaching materials.
*   **Projects:** A showcase of the projects I've worked on.
*   **Passions:** A glimpse into my hobbies and interests outside of work.
*   **Blog:** Thoughts and updates on various topics.

The website is built using [Hugo](https://gohugo.io/) and hosted on GitHub Pages.

Feel free to explore the live site at: [https://petruzzellialessandro.github.io/](https://petruzzellialessandro.github.io/)
